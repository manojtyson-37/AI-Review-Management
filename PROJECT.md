# ReviewAssist AI

**Tagline**: Turn Happy Customers into Authentic Google Reviews.

## Overview
ReviewAssist AI is an AI-powered reputation management SaaS designed to help businesses collect genuine Google Reviews. It focuses on a highly compliant, frictionless guest experience using AI to draft reviews based on short feedback.

## Tech Stack
*   **Frontend**: Next.js, React, TailwindCSS, ShadCN UI
*   **Backend**: NestJS, PostgreSQL (Prisma), Redis (BullMQ)
*   **AI**: OpenAI API (GPT-4o-mini for fast, cheap, high-quality generation)
*   **Infrastructure**: Turborepo (Monorepo), Docker, AWS S3

## Guiding Principles
*   **Scale**: Architected for 100k businesses, 10M reviews, 50M sessions.
*   **Compliance**: Strict adherence to Google Policies & FTC regulations (No review gating, no star manipulation).
*   **Design**: Premium, minimal, Apple-quality aesthetic.
*   **Automation**: Everything that can be automated, must be automated.
